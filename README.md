# haby 〜服薬・サプリ習慣化アプリ〜

---

## **サービス概要**

健康のために薬やサプリを続けたいけれど、つい飲み忘れてしまう人のための**服薬習慣化支援アプリ**です。

日々の服薬をワンタップで記録し、カレンダーで可視化することで「続けられる設計」をサポートします。

記録の手間を最小限に、かつ習慣の達成感を最大化するデザインを目指しています。

---

## **このサービスへの思い・作りたい理由**

私自身、以前から健康のためにサプリメントを飲むようにしていましたが、仕事が忙しい時期や体調が悪い時など、気づけば数日飲み忘れてしまうことがありました。

「続けることの大切さ」はわかっているのに、なかなか習慣化ができず日常の小さな忘れが積み重なってしまう。そのもどかしさを、何とかしたいと思ったのがきっかけです。

同時に、周囲でも「薬を飲み忘れてしまう」「サプリを買ったけど続かない」といった声を多く耳にしました。

「根性ではなく、設計や仕組みで習慣を変えられる」

そう考えたとき、心理的ハードルを下げるデザインと記録体験を組み合わせたアプリを作成しようとアイデアが生まれました。

| ユーザー層 | 理由 |
| --- | --- |
| **サプリメントの服用を習慣にしたい 20〜40 代の男女** | 美容・健康維持のために複数のサプリを摂取している層が多く、「続けたいけど忘れる」という課題が共通しているため。 |

---

## **サービスの利用イメージ**

1. ユーザーは自身が決めた時間にアプリを開き、「今日飲んだ」ボタンをワンタップ。
2. 「今日飲んだ」ボタンを押すと、アプリ内で小さな達成アニメーションやメッセージが表示される。
3. カレンダーにチェックが入り、服薬習慣が自動で記録される。
4. 継続の可視化によりモチベーションが高まり、自然と「飲むのが当たり前」になる。

→ **得られる価値**：

「続けられた」という実感が日々積み重なり、自己効力感と健康意識の向上につながる。

## **サービスの差別化ポイント・推しポイント**

| 差別化ポイント | 内容 |
| --- | --- |
| **① “記録のしやすさ”に特化した設計** | ワンタップで記録完了できる UI を採用し、記録の手間を最小化。続けやすさを最優先に設計。 |
| **② 継続したくなるデザイン設計** | 習慣達成時アニメーション・カレンダー UI・小さな成功体験の積み重ねで、“続けたくなる体験”を提供。 |
| **③ UI/UX へのこだわり** | 既存医療アプリ特有の堅さを排除し、生活の一部として溶け込む優しく柔らかなデザインを追求。 |
| **④ 失敗しても安心できる設計** | 一般的な習慣化アプリは、連続記録が途切れると萎えてやめてしまうことがあるが、haby は「習慣が途切れても気にしないメッセージ」やカレンダーで“長期的な継続”を俯瞰・重視できるように設計されている。 |

他アプリとの違い・推しポイント：

徹底的な記録のしやすさにこだわった設計。 “忙しい人ほど続けられる” 設計

### 既存アプリ比較：MyTherapy

医療寄りの服薬管理アプリで、

- 血圧・血糖値・体重・痛みなど多くの項目
- 医師に見せるためのレポート機能
- 治療計画を前提とした詳細なスケジュール設定

など機能がとても豊富です。

しかしその反面、

「ここまでしっかり管理するほどではないんだけど、サプリだけ続けたい」

という人には少し重く感じられることもあります。

---

haby はあえて機能を絞り込み、

- サプリ名とタイミングだけで始められる
- ログ項目は「服薬」「気分」など、最小限
- 医療データの細かい入力は不要
- MyTherapyにはない服薬達成時のアニメーション設計

というシンプルかつモチベーション向上を重視した **ミニマル設計** にすることで、

忙しい人でも **「とりあえず続けてみる」入口として使いやすい** ことを目指しています。

だからこそ、忙しい社会人・主婦・学生ほど続けやすくなる設計になっています。

→ “時間のない人の味方” という大きな価値訴求になる。

haby は、“達成感”と“シンプルな設計”を軸に「続けたくなる心理設計」で差別化しています。

## **機能候補**

### **MVP リリース（卒業制作範囲）**

- ユーザー登録 / ログイン（Devise）
- サプリ・薬の習慣登録・確認・編集・削除（CRUD）
- 日々の服薬記録（1日単位・チェック機能）
- カレンダーで服薬履歴を表示
- UI デザイン（Figma 反映 / Tailwind 導入）

### **本リリース（拡張予定）**

- プッシュ通知・リマインダー機能
- Google ログイン機能
- 週・月単位のグラフ表示
- 服薬率・継続日数の自動集計
- 習慣達成時のアニメーション演出
